# Brain-games
It is a set of five console inspired by popular mobile brain-pumping apps. Each game asks questions to be answered correctly. Once three correct answers are given, the game ends. Incorrect answers end the game and prompt you to play it again

## For install Brain Games
```sh
cd clone https://github.com/mgtsuu/frontend-project-44.git
cd frontend-project-44
npm link
```
